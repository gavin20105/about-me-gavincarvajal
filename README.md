# About Me

A simple personal "About Me" web page.

## Features

The page includes three interactive buttons, built with JavaScript:

- **Change Color** — turns the page heading navy when clicked.
- **Tell Me a Fact** — picks one of five fun facts about me at random and displays it on the page.
- **Click Counter** — starts at "Clicked 0 times" and counts up by one with every click.
